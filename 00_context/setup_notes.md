## フォルダの使い分け

| フォルダ | 用途 |
|---|---|
| `00_context/` | 会社概要・自己紹介・背景情報 |
| `01_strategy/` | 戦略・OKR・競合調査・週次レビュー |
| `02_finance/` | 売上・経費・請求書・税務（確定申告は`tax/`） |
| `03_projects/` | 議事録・案件資料・セミナー |
| `04_clients/` | 顧客・パートナー・見込み客 |
| `05_content/` | 記事・ブログ・SNS投稿（drafts/→articles/） |
| `docs/` | 社内資料・メンバー情報 |
| `~/Claude/projects/` | アプリ開発（コード系はここ） |

---

## 決定事項・ルール

- ファイル保存前に必ず保存先を確認する
- アプリ開発は`~/Claude/projects/プロジェクト名/`に保存
- 記事・コンテンツは`05_content/`に一元管理
- 確定申告などの税務は`02_finance/tax/`
